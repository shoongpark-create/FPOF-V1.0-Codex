# 라인시트 작성

> 컬렉션 플래너가 SKU별 상세 기획과 물량/사이즈/컬러 계획을 수립하는 전문 매뉴얼

## 언제 사용
- "라인시트 작성해줘", "물량 얼마나 잡아?", "사이즈 비율 정해줘"
- Plan 단계 마지막 작업 (MD 전략 완료 후)

## 사전 준비
1. `output/[시즌]/plan/season-concept.md` → MD 전략, 카테고리 믹스 (필수 선행)
2. `presets/wacky-willy/categories.json` → 카테고리별 아이템 구조
3. `presets/wacky-willy/channels.json` → 채널별 물량 배분 기준

## 실행 절차

### Step 1: SKU 리스트 작성
MD 전략의 카테고리 믹스를 개별 SKU로 전개한다.
- 스타일코드 부여 규칙: [시즌][라인][카테고리][번호] (예: 26SS-UNI-TOP-001)
- 각 SKU의 기본 정보: 아이템명, 라인, 카테고리, 가격대
- 챔피언 상품 표시 (★)
- Carry-over/리뉴얼 표시

### Step 2: OTB (Open-to-Buy) 계획
카테고리별 구매 예산을 설계한다.
- 목표 매출 → 원가율 → 마크업 → OTB 금액 산출
- 채널별 OTB 배분
- 시즌 초기 발주 vs QR 예비 비율

### Step 3: 사이즈 브레이크다운
아이템 유형별 사이즈 비율을 설정한다.
- 유니섹스: S/M/L/XL (또는 FREE)
- 우먼스: S/M/L (또는 FREE)
- 용품: FREE 또는 사이즈별
- 비율 기준: 전 시즌 판매 데이터 기반 (없을 경우 업계 평균)

### Step 4: 컬러 운영 계획
시즌 키 컬러를 아이템에 배분한다.
- 시즌 키 컬러 (visual-identity.json 기반)
- 아이템별 컬러웨이 수
- 베이직 컬러 vs 시즌 컬러 비율
- 컬러별 예상 판매 비중

### Step 5: 생산 일정 역산
런칭일 기준으로 생산 마일스톤을 역산한다.
- 런칭일 → 입고일 → 벌크 생산 → 샘플링 → 테크팩 완성 → 디자인 확정
- 각 단계별 소요 기간
- 크리티컬 패스 표시
