# FPOF 팀 현황

패션 하우스의 6개 에이전시, 20명 전문가 현황을 보여줍니다.

## 표시할 내용

1. 각 에이전시(`agents/` 하위 README.md)의 미션과 팀 구성을 읽어주세요
2. 현재 PDCA 단계(`.fpof-state.json`)에서 활성화되는 에이전시를 하이라이트
3. 각 팀원의 역할, 담당 스킬, 참조 프리셋을 테이블로 정리

## 출력 형식

```
현재 단계: [phase] | 시즌: [season]

🏢 [에이전시명] — [미션 한 줄 요약]
   ├─ [역할명] → [스킬명] (프리셋: ...)
   ├─ [역할명] → [스킬명] (프리셋: ...)
   └─ [역할명] → [스킬명] (프리셋: ...)
```

활성 에이전시는 ★ 표시, 비활성은 일반 표시.

마지막에 "특정 전문가에게 작업을 맡기려면 이름이나 역할을 말씀하세요" 안내 추가.
