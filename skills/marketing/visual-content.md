# 비주얼 콘텐츠 기획

> 콘텐츠 디렉터가 화보/영상 등 마케팅 비주얼 콘텐츠를 기획하는 전문 매뉴얼

## 언제 사용
- "화보 촬영 기획해줘", "영상 콘텐츠 만들자", "숏폼 기획해줘"
- Do 단계에서 IMC 전략 수립 후

## 사전 준비
1. `output/[시즌]/do/campaign-brief.md` → 캠페인 방향, 3단계 로드맵
2. `output/[시즌]/design/[아이템]-moodboard.md` → 시즌 비주얼 톤
3. `presets/wacky-willy/visual-identity.json` → 브랜드 비주얼 가이드
4. `presets/wacky-willy/tone-manner.json` → 브랜드 톤
5. `presets/wacky-willy/ip-bible.json` → 캐릭터 활용 시

## 실행 절차

### Step 1: 화보 촬영 기획
- 컨셉: 시즌 테마와 연결된 비주얼 스토리
- 모델: 뮤즈 지젤 / 일반 모델 / 인플루언서
- 로케이션: 스튜디오 / 야외 / 특수 장소
- 스타일링 방향: 단품 / 코디네이트 / 레이어드
- 컷 리스트: 전신/반신/디테일/무드 각 몇 컷

### Step 2: 영상 콘텐츠 기획

| 포맷 | 플랫폼 | 길이 | 목적 |
|------|--------|------|------|
| 숏폼 리얼 | TikTok | 15~30초 | 바이럴, 참여 유도 |
| 숏폼 무드 | 인스타 릴스 | 15~60초 | 감성 전달 |
| 캠페인 필름 | 유튜브/인스타 | 30~60초 | 브랜드 스토리 |
| 비하인드 | 인스타/TikTok | 15~30초 | 친밀감 |

### Step 3: 채널별 에셋 기획

| 채널 | 에셋 유형 | 포맷 | 사이즈 | 수량 |
|------|----------|------|--------|------|
| 인스타 피드 | 화보 이미지 | JPG | 1080×1080 | |
| 인스타 스토리 | 이미지/영상 | MP4/JPG | 1080×1920 | |
| TikTok | 숏폼 영상 | MP4 | 1080×1920 | |
| 자사몰 | 배너/PDP | JPG | 다양 | |
| 외부몰 | 썸네일/상세 | JPG | 플랫폼별 | |

### Step 4: 크리에이티브 스튜디오 협업
- AI 이미지 생성 요청 사항
- 실촬영 vs AI 생성 구분
- 비주얼 검수 기준
