# FPOF PDF 보고서 생성

산출물을 정리하여 전문적인 PDF 보고서로 변환합니다. PDF 스킬을 활용합니다.

## 사용법
`/pdf [유형]` — 유형을 생략하면 현재 단계의 전체 보고서를 생성합니다.

## PDF 유형

### 시즌 문서
- `trend-report` — 트렌드 분석 보고서 (리서치 결과, 차트, 시사점)
- `season-book` — 시즌 북 (컨셉, 디자인, 상품 구성 통합 문서)
- `line-sheet` — 바이어용 라인시트 (아이템별 이미지, 스펙, 가격)

### 생산 문서
- `techpack` — 테크팩 PDF (도식화, BOM, 사이즈 스펙, 봉제 사양)
- `cost-sheet` — 원가 산출서 (아이템별 원가 명세)

### 마케팅 문서
- `campaign-brief` — 캠페인 브리프 (전략, 타임라인, 예산)
- `press-kit` — 프레스 키트 (브랜드 스토리, 시즌 하이라이트, 이미지)

### 분석 보고서
- `season-report` — 시즌 종합 보고서 (Plan~Check 전 과정 요약)
- `kpi-dashboard` — KPI 대시보드 보고서

## 절차

1. `output/[시즌]/`에서 관련 산출물 수집
2. `presets/wacky-willy/visual-identity.json`에서 브랜드 스타일 참조
3. PDF 스킬을 사용하여 문서 생성:
   - 표지: 브랜드 로고 영역, 시즌명, 문서 제목, 날짜
   - 목차 자동 생성
   - 본문: 산출물 내용을 보고서 형식으로 재구성
   - 헤더/푸터: 브랜드명, 페이지 번호, 보안 등급
4. `output/[시즌]/[프로젝트]/` 에 PDF 파일 저장

## 디자인 원칙
- 브랜드 컬러 팔레트 적용
- 깔끔한 레이아웃, 여백 충분히
- 차트/테이블은 가독성 우선
- 이미지 해상도 유지

$ARGUMENTS 에 유형이 지정되면 해당 PDF를, 없으면 현재 단계 전체 보고서를 생성합니다.
