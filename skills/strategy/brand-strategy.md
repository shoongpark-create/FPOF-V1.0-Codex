# 브랜드 전략 수립

> 브랜드 전략가가 트렌드를 와키윌리의 시각으로 해석하고 시즌 방향성을 정의하는 전문 매뉴얼

## 언제 사용
- "시즌 테마 뭘로 하지?", "브랜드 방향 잡아줘", "포지셔닝 검토해줘"
- Plan 단계에서 트렌드 리서치 완료 후 진행

## 사전 준비
1. `output/[시즌]/plan/trend-brief.md` → 트렌드 리서치 결과 (필수 선행)
2. `presets/wacky-willy/brand.config.json` → 브랜드 DNA, 비전, 5대 경영목표
3. `presets/wacky-willy/personas.json` → UNI/WOMAN 페르소나
4. `presets/wacky-willy/visual-identity.json` → 비주얼 아이덴티티 기준
5. `knowledge/` → 이전 시즌 브랜드 전략 리뷰 (있을 경우)

## 실행 절차

### Step 1: 시즌 테마 도출
트렌드 브리프의 추천 키워드를 와키윌리 세계관으로 번역한다.
- 트렌드 키워드 → 와키윌리 고유 언어로 재해석
- 브랜드 DNA(Kitsch Street & IP Universe)와의 연결점 찾기
- 시즌 테마 후보 2~3개 → 최종 1개 선정
- 테마 네이밍: 와키윌리답게 위트 있는 이름

### Step 2: 포지셔닝 확인/조정
현재 포지셔닝이 유효한지 점검한다.
- 유니섹스 라인: "자유분방한 스트릿 with IP 유니버스" 유지 여부
- 우먼스 라인: "시크한 유니크 캐주얼" 유지 여부
- 경쟁사 대비 차별점이 명확한가?
- 가격 포지셔닝 적정성 (코어 타겟의 구매력 기준)

### Step 3: 경영목표 연결
시즌 테마가 5대 경영목표 중 어떤 것에 기여하는지 명시한다.
1. 브랜드 아이덴티티 정립 → 코어타겟 매출 비중 확대 방안
2. 히트상품 + IMC 강화 → 상위 20% 집중 전략
3. QR 비중 확대 → 인시즌 대응력
4. 용품 라인업 경쟁력 → 용품 카테고리 강화 방안
5. 글로벌 대응 강화 → 해외 시장 타겟팅

### Step 4: 타겟 업데이트
이번 시즌의 타겟 포커스를 정한다.
- 메인 타겟 유지 확인 (18~25세 자유로운 트렌드리더)
- 시즌별 포커스: UNI 중심 / WOMAN 확대 / 글로벌 등
- 페르소나 업데이트 (계절/트렌드 반영)

### Step 5: IP 활용 방향
시즌 테마에 맞는 캐릭터 활용 전략을 수립한다.
- 메인 캐릭터 선정 (키키 + 누구?)
- 시즌 스토리라인 (키키의 어떤 모험/변신?)
- 그룹별 활용: A그룹(메인) / B그룹(그래픽) / C그룹(서브)
- 신규 변신/콜라보 컨셉
