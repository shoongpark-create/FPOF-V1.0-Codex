# 무드보드 제작

> 크리에이티브 디렉터가 시즌 컨셉을 시각 언어로 번역하고, 전체 디자인의 톤을 설정하는 전문 매뉴얼

## 언제 사용
- "무드보드 만들어줘", "비주얼 방향 잡아줘", "레퍼런스 보여줘"
- PDCA Design 단계 첫 번째 작업

## 사전 준비
1. `output/[시즌]/plan/brand-strategy.md` → 시즌 테마, IP 활용 방향
2. `output/[시즌]/plan/season-concept.md` → 키 컬러, 키 실루엣, 키 소재
3. `presets/wacky-willy/visual-identity.json` → 브랜드 비주얼 가이드
4. `presets/wacky-willy/ip-bible.json` → 캐릭터 비주얼/성격

## 실행 절차

### Step 1: 시즌 비주얼 키워드 추출
시즌 테마를 3~5개의 비주얼 키워드로 변환한다.
- 시즌 컨셉에서 감성 키워드 추출
- visual-identity.json의 디자인 키워드(Doodle/Graffiti/Pop Art)와 교차
- 레퍼런스 이미지 수집 방향 결정

### Step 2: 컬러 팔레트 구성
시즌 컬러를 구체적으로 정의한다.
- 메인 컬러 (2~3개): 시즌을 대표하는 색
- 서브 컬러 (2~3개): 보조 색상
- 뉴트럴 (1~2개): 베이스 색상
- visual-identity.json의 시그니처 컬러(비비드 오렌지, 블랙, 화이트)와의 조화

### Step 3: 텍스처/소재 무드
시즌 핵심 소재의 질감과 무드를 정의한다.
- 소재 카테고리 (저지/데님/나일론/니트 등)
- 질감 키워드 (가벼운/무거운/광택/매트/빈티지 등)

### Step 4: 실루엣/핏 레퍼런스
시즌 키 실루엣을 정의한다.
- 오버사이즈/레귤러/슬림 등 핏 방향
- 기장 트렌드 (크롭/롱/미디 등)
- 카테고리별 실루엣 가이드

### Step 5: 그래픽/IP 캐릭터 활용 무드
IP 캐릭터와 그래픽 스타일을 무드로 정의한다.
- 캐릭터 활용 스타일: 리얼/SD/아이콘화/패턴화
- 그래픽 기법: 프린트/자수/와펜/패치/올오버
- 시즌 스토리와 연결된 캐릭터 장면/포즈

### Step 6: 라이프스타일 이미지
타겟 고객의 일상 속 착장 무드를 정의한다.
- 어떤 장소에서 입는가? (스트릿/캠퍼스/페스티벌/카페)
- 어떤 액티비티와 연결되는가?
- 스타일링 무드 (믹스매치/레이어드/미니멀 등)
