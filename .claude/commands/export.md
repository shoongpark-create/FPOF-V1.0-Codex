# FPOF 산출물 내보내기

현재 시즌의 산출물을 정리하여 요약합니다.

## 절차

1. `.fpof-state.json`에서 현재 시즌 확인
2. `output/[시즌]/` 하위 모든 파일을 스캔
3. PDCA 단계별로 산출물 목록 정리:

### 출력 형식
```
📦 [시즌코드] 산출물 현황
━━━━━━━━━━━━━━━━━━━━━━━━━

📋 Plan (기획)
   ✅ trend-brief.md — 트렌드 분석 보고서
   ✅ brand-strategy.md — 시즌 테마/전략
   ❌ md-plan.md — (미작성)
   ...

🎨 Design (디자인)
   ...

🔨 Do (실행)
   ...

📊 Check (분석)
   ...

━━━━━━━━━━━━━━━━━━━━━━━━━
전체: X/Y 완료 (Z%)
```

4. 각 파일의 첫 5줄(제목/요약)을 읽어 간단한 설명 추가
5. 누락된 산출물은 해당 스킬 파일에서 필수 여부 확인 후 표시

$ARGUMENTS 가 "detail"이면 각 산출물의 주요 내용까지 요약합니다.
