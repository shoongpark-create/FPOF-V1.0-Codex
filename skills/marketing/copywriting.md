# 카피라이팅 & 스토리텔링

> 패션 에디터가 브랜드 보이스로 모든 텍스트 콘텐츠를 작성하는 전문 매뉴얼

## 언제 사용
- "상품 설명 써줘", "캡션 만들어줘", "스토리 정리해줘"
- Do 단계에서 상품/캠페인 카피 작업 시

## 사전 준비 (필수!)
1. `presets/wacky-willy/tone-manner.json` → **반드시 참조** (모든 카피의 톤 기준)
2. `presets/wacky-willy/ip-bible.json` → 캐릭터 스토리 연결 시
3. `presets/wacky-willy/personas.json` → 타겟 언어/감성
4. `output/[시즌]/design/[아이템]-design-spec.md` → 상품 정보

## 톤앤매너 사전 체크 (매번 확인!)
- [ ] 유쾌하고 자유분방한 톤? (경직된 어투 X)
- [ ] Gen-Z 친화적 언어? (과도한 존댓말 X)
- [ ] 브랜드 필수 어휘 포함? (와키윌리, KIKY 등)
- [ ] 금지 어휘 미사용? (저렴한, 짝퉁, 따라하기, 평범한)
- [ ] 채널에 맞는 톤과 길이?

## PDP 카피 구성
1. **상품명** (국문/영문) — 브랜드 네이밍 룰 적용
2. **한 줄 후크** — 스크롤을 멈추게 하는 첫 문장
3. **상품 스토리** — 3~5문장, 시즌 테마/IP 세계관과 연결
4. **핵심 셀링 포인트** — 3개, 기능+감성 밸런스
5. **소재/케어 정보**
6. **사이즈 가이드 한 줄**
7. **추천 스타일링 팁**

## SNS 채널별 카피

**인스타그램 피드:**
- 캡션: 감성적, 2~3줄
- 해시태그: 한글+영문 믹스, 10~15개
- CTA: 좋아요/댓글/저장 유도

**TikTok:**
- 텍스트 오버레이: 1~2줄, 임팩트
- 캡션: 짧게, 참여 유도
- 해시태그: 트렌드 태그 포함, 5~8개

**자사몰 배너:**
- 헤드라인: 1줄, 강렬하게
- 서브카피: 1줄, 부연 설명
- CTA 버튼 문구: 행동 유도
