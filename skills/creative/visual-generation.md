# AI 비주얼 생성

> 아트 디렉터가 디자인 스펙을 기반으로 AI 이미지를 생성하고 비주얼 품질을 관리하는 전문 매뉴얼

## 언제 사용
- "이미지 만들어줘", "룩북 생성해줘", "캠페인 비주얼 작업해줘"
- Design 단계에서 디자인 스펙 완료 후 진행

## 사전 준비
1. `output/[시즌]/design/[아이템]-design-spec.md` → 디자인 상세 (필수 선행)
2. `output/[시즌]/design/[아이템]-moodboard.md` → 비주얼 톤/무드
3. `presets/wacky-willy/visual-identity.json` → 브랜드 비주얼 스타일
4. `presets/wacky-willy/ip-bible.json` → 캐릭터 비주얼 참조

## 실행 절차

### Step 1: 비주얼 유형 결정
| 유형 | 용도 | 스타일 | 배경 |
|------|------|--------|------|
| 플랫 이미지 | 라인시트, 테크팩, PDP | 화이트 배경, 정면/후면 | 클린 |
| 룩북 이미지 | 시즌 프레젠테이션 | 모델 착장, 무드 배경 | 스트릿/스튜디오 |
| 디테일 이미지 | PDP, 상품 소개 | 소재/디테일 클로즈업 | 중립 |
| 캠페인 이미지 | 마케팅, SNS | 무드 중심, 브랜드 감성 | 콘셉추얼 |
| 배너 이미지 | 자사몰, 외부몰 | 텍스트 공간 확보 | 브랜드 컬러 |

### Step 2: AI 프롬프트 설계
디자인 스펙을 AI 이미지 생성 프롬프트로 변환한다.
- 주체: 모델 정보 (연령대, 체형, 포즈)
- 의상: 디자인 스펙의 실루엣/소재/컬러/그래픽
- 스타일: 무드보드의 감성 키워드
- 배경: 유형별 배경 설정
- 라이팅: 자연광/스튜디오/무드 조명
- 브랜드 톤: Kitsch Street, Pop Art, Doodle/Graffiti 감성

### Step 3: 이미지 생성 & 검수
- [ ] visual-identity.json의 컬러 팔레트가 정확히 반영되었는가?
- [ ] Kitsch Street 감성이 유지되고 있는가?
- [ ] IP 캐릭터 사용 시 ip-bible.json의 외형/성격 설정을 따르는가?
- [ ] 18~25세 타겟이 공감하는 스타일링인가?
- [ ] 의류 디테일이 디자인 스펙과 일치하는가?

### Step 4: 채널별 최적화
| 채널 | 비율 | 사이즈 | 특이사항 |
|------|------|--------|---------|
| 인스타 피드 | 1:1 | 1080×1080 | 캡션 공간 |
| 인스타 스토리 | 9:16 | 1080×1920 | 스와이프 유도 |
| TikTok | 9:16 | 1080×1920 | 텍스트 오버레이 |
| 자사몰 PDP | 다양 | 상품 기준 | 화이트 배경 |
| 배너 | 다양 | 채널별 상이 | 텍스트 영역 확보 |
